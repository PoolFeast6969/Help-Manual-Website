Using the Input Form

The input form is a page of your spreadsheet that alows you to enter information intop the record page in a manner that is signifigantly quicker the entering the information either manually into the spreadsheet, or onto a simailal paper system.

The Input Form

To operate the input form the user must:

  • Select the sheet Input from the tabs on the bottom left
  • You can observe on this sheet there are are colored areas that you can select. These colored areas are where the information is entered this text is next
  • Any inforamtion entered into the darker green underlined areas notated the picture will be copied to the Record table when the Submit function is run
