A few basic operating pricipals:

  • Sheets are different pages or views of the same spreadsheet file. The different functions of this Spreadsheet file have been split into 6 sheets, however, only three have to be accessed to fulfill basic operation.
  • To switch between sheets, which have names that describe their function such as Input for Input Form, the user can select them by clicking on the tabs as the bottom left of the Excel interface. As seen in Figure 1
  • The cursor is the easiest method of interraction with Excel, if this Manual asks you to select or interact with some element of the screen, using the cursor the unspoken assumption.
Using the Input Form

The input form is a page of your spreadsheet that allows you to enter information into the record page in a manner that is significantly quicker the entering the information either manually into the spreadsheet, or onto a similar paper system.

Fig 2 - The Input Sheet

To operate the input form the user must:

  • Select the sheet Input from the tabs on the bottom left.
  • You can observe on this sheet, Figure 2, there are are colored areas that you can select. These colored areas are where the information is entered this text is next.
  • Any information entered into the darker green underlined areas notated the picture will be copied to the Record table when the Submit function is run.
